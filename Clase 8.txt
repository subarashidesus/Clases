1. Construir un programa que pida ingresar un número positivo y muestre el número de cifras de este.


2. Desarrollar un pseudocódigo que luego de ingresar 2 números naturales imprimar los números naturales que hay entre ambos empezando por el menor. Asumimos que el usuario ingresa dos números diferentes y el segundo es mayor que el primero. Emplear la estructura repetitiva MIENTRAS.

3. Calcular el precio total de una compra de "n" productos ( el valor "n" se ingresa por teclado) si se ingresa el precio de cada producto y la cantidad comprad de este. Emplear la estructura repetitiva MIENTRAS.

4. Se desar crear un algoritmo para un sistema de votación. Supóngase en este caso que hay cuatro candidatos, con identificadores 1, 2, 3, 4. Usted habrá de calcular mediante un programa, el número de votos correspondiente a cada candidato. El usuario ingresara los votos de manera desorganizada, tal y como se obtuvieron en la elección. El conteo de votos se detiene si se ingresa el valor cero. Observe, como ejemplo, la siguiente lista: 1 3 1 4 4 1 2 4 4 0

5. Construir un pseudocódigo que solo permite ingresar números positivos, en caso de que el número no sea positivo debe volver a pedir que se ingrese el número hasta que el número hasta que el usuario ingrese un número positivo, luego mostrar el número.

6. Se lanza un dado reiteradas veces hasta que el valor del tiro es 1. Deternubar la cantidad de veces que se lanzó el dado y el puntaje acumulado, si lo que se acumula es el valor del tiro.


7. Desarrollar un algoritmo que calcule el promedio de los primeros 10 números naturales usando la estructura REPETIR - HASTA QUE

8. Construir un pseudocódigo que valide la nota de un estudiante, el programa debe pedir la nota y solo si esta se encuentra en el rango de 0 a 20 se debe mostrar. Debe pedir la nota mientras se encuentra fuera de ese rango.


9. Construir un algoritmo que permita ingresar un número "n" de estudiantes (se cumple que: 1 <= n <= 50 ), cada uno de los cuáles cursa 5 materias. Para cada alumno se ingresa, clave y las 5 calificaiones. Escriba un algoritmo para calcular el promedio de cada estudiante luego de ingresar sus notas.
