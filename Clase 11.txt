1. Ejemplo de recorrido de un array unidimensional en java
Programa que lee por teclado la nota de los alumnos de una clase y calcula la nota media de un grupo. También muestre los alumnos con notas superiores a la media. El número de alumnos se lee por teclado.

Este programa crea un array de elemento de tipo double que contradrá las notas de los alumnos. El tamaño del array será el número de alumnos de la clase.


Se realizan 3 recorridos sobre el array, el primero para asignar a cada elemento las notas introducidas por teclado, el segundo para sumarlas y el tecero para mostrar los alumnos con notas superiores a la media.
